该项目的功能需求如下：
app用户是环球旅行的人,从我自己的生活方式出发,我去过很多国家和城市,每到一个地方我都会打卡好看的咖啡店/书店/博物馆/美术馆/展览/买手店等 我手机里拍过很多照,但是没有一个平台能够完全承载这些记录 有些人喜欢喝酒的话,还可以加上bar,总之就是泛探店类的,不局限于咖啡赛道 暂时把这个定位成电子手账吧，或者后面可以跟记账一样分类,咖啡店/博物馆/书店/酒吧/餐厅

  - 自定义类型同步到首页筛选 chip

  我的页面
  - 头像支持从相册选图，本地持久化
  - 用户名可点击编辑，保存后生效
