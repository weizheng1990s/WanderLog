这是为环球探索爱好者，开发的一个电子记账手册

关注小红书账号：Catz_1201、mars_333

该项目的功能需求如下：
app用户是环球旅行的人,从我自己的生活方式出发,我去过很多国家和城市,每到一个地方我都会打卡好看的咖啡店/书店/博物馆/美术馆/展览/买手店等 我手机里拍过很多照,但是没有一个平台能够完全承载这些记录 有些人喜欢喝酒的话,还可以加上bar,总之就是泛探店类的,不局限于咖啡赛道 暂时把这个定位成电子手账吧，或者后面可以跟记账一样分类,咖啡店/博物馆/书店/酒吧/餐厅



3月20日完成的功能：
  首页
  - ✦ WANDER 改为 ✦ Kiro Book
  - 打卡列表改为月历视图，有打卡的日期显示类型图标
  - 日历支持左右滑动切换月份
  - 首页名字/签名与「我的」页面联动

  日历交互
  - 点击有打卡的日期 → 全屏弹出当天所有打卡
  - 每条打卡以卡片风格展示（大图 + 店名）
  - 多条打卡垂直列表排列，可滚动
  - 点击卡片进入详情页

  打卡页面（新增类型）
  - 「其他」按钮改为「+」新增按钮
  - 支持自定义类型的新增 / 编辑 / 删除
  - 长按自定义类型显示操作菜单
  - 自定义类型同步到首页筛选 chip

  我的页面
  - 头像支持从相册选图，本地持久化
  - 用户名可点击编辑，保存后生效
  - 个性签名可点击编辑，保存后生效
  - 首页 Hello / Explorer 同步显示用户名和签名
