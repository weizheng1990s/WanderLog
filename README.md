这是为环球探索爱好者，开发的一个电子记账手册

app用户是环球旅行的人,从我自己的生活方式出发,我去过很多国家和城市,每到一个地方我都会打卡好看的咖啡店/书店/博物馆/美术馆/展览/买手店等 我手机里拍过很多照,但是没有一个平台能够完全承载这些记录,
